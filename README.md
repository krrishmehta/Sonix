# Sonix
Sonix is a real-time music synchronization web app that allows multiple users to listen to music together from different locations. Inspired by Spotify Jam, it syncs play, pause, seek, and track changes across all participants using FastAPI and WebSockets, delivering a seamless, low-latency shared listening experience.
